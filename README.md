# SmartFlow AI CRM Assistant

## AI-Powered Business Automation Assistant

SmartFlow AI CRM Assistant is a business automation system developed using Streamlit and SQLite. The project helps businesses automate customer interaction, lead management, analytics, and workflow handling through an intelligent assistant interface.

---

# Features

## AI Assistant

* Business query response system
* Automated assistant replies
* Chat history management

## Lead Capture System

* User lead submission form
* Customer information storage
* Interest category selection

## Lead Priority System

* High / Medium / Low priority classification
* Business lead scoring logic

## Analytics Dashboard

* Total leads tracking
* High priority lead tracking
* Interactive charts using Plotly

## Admin Dashboard

* Lead viewing interface
* CSV export functionality
* Data management system

## Automation Workflow

* Automated thank-you email system
* Auto lead summary generation
* Chat history logging

---

# Technology Stack

* Python
* Streamlit
* SQLite
* Pandas
* Plotly

---

# Database Used

SQLite Database

Tables:

* leads
* chat_history

---

# Project Workflow

1. User interacts with AI Assistant
2. User submits lead form
3. Data stored in SQLite database
4. Lead priority generated
5. Analytics dashboard updates automatically
6. Admin can export data as CSV
